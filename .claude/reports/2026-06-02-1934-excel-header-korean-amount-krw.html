엑셀 출력 — 헤더 한글화 + 결제금액 원화 표시

2026-06-02 19:34

.xlsx 헤더를 DB 컬럼명에서 한글로 바꾸고, 결제금액을 원화 단위로 표시되도록 셀 서식을 적용했다.

작업 요청 요약

  • 엑셀 헤더가 컬럼명(id, user_name, ...)으로 저장되던 것을 한글로 변경.
  • amount(결제금액)를 한화(원) 단위로 표시되게 저장.
  • 현재 브랜치 feat/excel-generation 미커밋 작업에 이어서 — 커밋하지 않음.

변경 파일

api/exceljob/excel/OrderExcelWriter.java ~ 헤더 한글화 + 결제금액 셀 서식(원화)

변경 내용

  • 헤더: 주문번호 / 주문자 / 상품명 / 카테고리 / 결제금액 / 주문상태 / 주문일시
  • 결제금액: 값은 숫자형 유지, 표시만 원화. CellStyle dataFormat #,##0"원" → 240,959원. Excel에서 합계·정렬 가능.
  • SXSSF 특성상 스타일은 write() 에서 1개만 생성해 전 행이 재사용(행마다 생성 금지).

검증

  • ./gradlew compileJava → BUILD SUCCESSFUL.
  • 런타임(백엔드 재기동 + MySQL/seed): POST로 생성 → 엑셀 열어 헤더 한글·결제금액 …원 표시 확인(사용자).

비고

  • 헤더 한글 라벨은 제안값 — 다른 표현 원하면 조정 가능.
  • API/스키마 변경 없음(엑셀 내부 표현만). openapi·테스트 영향 없음.
  • 커밋하지 않음.
